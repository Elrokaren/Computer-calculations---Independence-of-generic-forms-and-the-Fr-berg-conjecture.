# Computer-calculations---Independence-of-generic-forms-and-the-Fr-berg-conjecture.
This repository contains Macaulay2 used used to verify the Fröberg conjecture in some cases.  

The name of each file describes the number of variables n, and the finite field, Z/Zp,  which the computation was done over. In all cases the forms are of degree d, and the computatiom checks that Hilbert function in degree d+2. The code was used in the work Independence of generic forms and the Fröberg conjecture, see (put arXiv link).
